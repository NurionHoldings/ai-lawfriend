# AI법친 무료 유입 서비스 1·2호 OG 이미지 최종 기준

## 공통 규격

- 크기: 1200 x 630
- 배경: 어두운 네이비 또는 딥 블루 계열
- 글자: 모바일 미리보기에서 2초 안에 읽히는 큰 글자
- 로고: AI법친 또는 AI법친 심볼
- 법률판단·승소·구제 보장 표현 금지

## 시각 QA 체크리스트

- [ ] 카카오톡 공유 미리보기에서 제목이 잘리지 않는다.
- [ ] 네이버/페이스북/인스타 링크 미리보기에서 핵심 문구가 읽힌다.
- [ ] 모바일 화면에서 메인 문구가 2줄 이내로 보인다.
- [ ] 법률대리·승소·구제 보장처럼 오해될 표현이 없다.
- [ ] 두 서비스가 서로 다른 색감 또는 아이콘으로 구분된다.
