# AI법친 전세사기·보증금 반환 피해 서류 정리센터 A/B 테스트 문구

## 목적

전세사기·보증금 반환 피해 서류 정리센터의 SNS 광고·랜딩페이지·팝업 문구를 비교하여 서류 정리 시작률과 제출 성공률을 높인다.

## 공통 준수 기준

- "전세사기 피해자로 인정됩니다" 표현 금지
- "보증금을 받을 수 있습니다" 표현 금지
- "승소 가능성이 높습니다" 표현 금지
- "무료 법률상담" 표현 금지
- "서류 정리 보조", "피해사실 요약서", "제출자료 체크리스트" 표현 사용
- 최종 제출 전 공식기관 또는 변호사 검토 권장
