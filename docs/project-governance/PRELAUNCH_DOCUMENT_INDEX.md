**관련**

- 총괄 기준선: `docs/project-governance/TOP3_FINAL_INTEGRATED_JUDGMENT.md` §0-1
- 총괄 증빙: `docs/project-governance/IMPLEMENTATION_EVIDENCE.md` [EVIDENCE-20260423-262]

## 0. 이 문서의 목적

이 문서는 AI법친 공표·발표 직전에 사용하는 문서들을 **한 장에서** 찾고,  
상황에 따라 **어떤 문서를 먼저 열지** 빠르게 안내하기 위한 **인덱스**다.

세부 판정은 각 **개별 판정서**와 **증빙**에 유지하고,  
**공표용** **운영·발표·질의응답** 문서는 **이 인덱스**를 **출발점**으로 쓴다.

- 총괄 판단은 `TOP3_FINAL_INTEGRATED_JUDGMENT.md` §0-1 + [262]로 본다.
