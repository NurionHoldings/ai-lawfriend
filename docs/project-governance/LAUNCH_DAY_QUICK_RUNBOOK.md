## 지금: 실행 단계 (한 줄)

### 읽는 기준 (이 런북·직후를 이렇게 읽는다)

- **맨 위** 이 절: 지금부터 끝까지 **전체 실행 흐름**(파이프라인 한 줄).
- **「지금 해야 할 일」** 이하: **공표 당일** 집중 구간 — 오늘 바로 해야 할 것.
- **공표 이후** 미세 안정화: `POSTLAUNCH_STABILIZATION.md` **§0**·**§1**.

실행 **흐름·해석**의 공식 증빙은 `IMPLEMENTATION_EVIDENCE.md` **[264]** 이다. (**[263]** 은 본 문서 **당일 잠금** 증빙.)

**AI법친은 이제** 리허설 → 공표 → 직후 안정화까지의 **실행 파이프라인**이 문서와 증빙에 잠겼고, 실제로 따라가면 되는 단계다.

### 문서 쪽에서 이미 닫힌 것 (전체 흐름)

- **공표 전 준비** **문서 묶음** **완료**
- **총괄 기준선** **완전 잠금** **완료** (`TOP3_FINAL_INTEGRATED_JUDGMENT.md` §0-1 · **[262]**)
- **당일 실행 문서** **잠금** **완료** (**[263]** · 본 문서)
- **공표 직후 안정화** **문서 연결** **완료** (`POSTLAUNCH_STABILIZATION.md`)

**다음은 정말 문서 작업이 아니라 실행만 하면 된다.** 순서도 **이미 충분히** 명확하다.

1. **리허설** 1회  
2. **공표 실행** 1회  
3. **직후 메모** 4가지 기록  
4. **미세 수정**만 반영  

**같은 순서의 상세 체크**는 아래 **1)~4)** 를 따른다.

### 1) 리허설 1회

- `PRELAUNCH_DOCUMENT_INDEX.md`
- `LAUNCH_DAY_QUICK_RUNBOOK.md` (본 문서)
- `PRELAUNCH_DEMO_FLOW_ONEPAGE.md`
- `PRELAUNCH_DEMO_SPEECH_SCRIPTS.md`
- `PRELAUNCH_FAQ_10.md`

### 2) 공표 실행

- **대표 사건**으로 **안정적으로** 시연한다.
- **3분 대본**을 기준으로 말한다.
- 질문은 **FAQ 톤**대로 **2~4문장**으로만 답한다.

### 3) 공표 직후 기록 (이 4가지만 바로 적는다)

- 많이 나온 질문
- 막힌 설명
- 머뭇거린 화면
- 바꿔야 할 문구

### 4) 직후 안정화

- `POSTLAUNCH_STABILIZATION.md` 기준으로 진행한다.
- **기능 추가 없이** — **문구·안내·실패 메시지·데모 동선**만 **미세 수정**한다.

- **공표 당일 한 줄:** **§4**·**§6**·아래 문장과 같이 본다.

## 11. 문서 잠금 · 이번 반영

### 잠금 한 줄

`LAUNCH_DAY_QUICK_RUNBOOK.md`는 **당일 리허설**과 **공표 실행**의 **우선순위**, **기준 문서**, **실행 원칙**까지 **바로 따라갈 수 있는** **당일 실행 기준 문서**로 **잠겼다**.

### 이번 반영으로 좋아진 점

- **문서 맨 앞**에서 **「지금 해야 할 일」2가지**가 **바로** 보여 **당일 판단**이 **빨라짐**.
- **리허설**과 **공표 실행**이 **실행 순서** + **기준 문서** + **원칙**까지 **한 번에** 묶임.
- **공표 당일 한 줄**(**§4**·**§6**·**맨 앞** **확정 문장**)이 **고정**되어, **흔들릴 때** **다시 중심**을 **잡기** **쉬움**.

### 전체 흐름 닫힘 (실행 전환)

- 본 문서는 **당일 실행 기준 문서**로 **잠겼고**, 증빙은 **[263]** 이다.
- **준비·기준·당일·직후** **네 축**이 **문서상** **완료**됐으므로, **남은 것은 실행뿐**이다 — **압축 순서:** **리허설 1회 → 공표 1회 → 직후 메모 4가지 → 미세 수정만** (**§** 맨 앞).

### 실행 파이프라인 해석 잠금 ([264])

- **전체 실행 파이프라인**과 **공표 당일 집중 구간**이 문서 구조상 **분리**되어 덜 헷갈린다 — **맨 앞 절** / **「지금 해야 할 일」** / **`POSTLAUNCH` §0·§1** (위 **「읽는 기준」**).
- 실행 단계 **해석**까지 잠긴 것의 **공식 기록**은 **[264]** 이다.
