# AI법친 6.9 — 개인정보 / 보안 / 동의문구 최종 정리

## 1. 목적

본 문서는 AI법친 6.9의 기준문서다.

6.0~6.8에서 구현한 사건 패키지 / 고유번호 공유 / 변호사 열람 / 첨부 다운로드 / 요약본 출력 기능에 대해 개인정보, 보안, 공유 동의, 로그 보관, 출력물 제외 기준, 변호사법 오인 방지 문구를 최종 정리한다.

이번 단계는 신규 Prisma schema 변경이나 API 확장이 아니라, 기존 구현에 적용되는 보안·고지·동의·출력 정책을 문서와 공용 상수/유틸로 고정하는 단계다.

## 4. 의뢰인 공유 동의문구

의뢰인이 사건 패키지를 공유할 때 아래 동의문구를 기준으로 한다.

```text
본인은 선택한 사건 정보, 사건 요약, AI 인터뷰 요약, 첨부자료 목록 및 문서 초안의 기초가 지정한 변호사 또는 전문가에게 제공될 수 있음을 확인합니다.

본인은 공유 범위, 첨부파일 다운로드 허용 여부, 사건 패키지 요약본 다운로드 허용 여부, 공유 기간을 확인했습니다.

본인은 사건 패키지 공유 후 변호사 또는 전문가의 열람·다운로드 기록이 시스템에 저장될 수 있음을 확인합니다.

본인은 공유를 언제든 취소할 수 있으며, 공유가 취소되거나 만료된 이후에는 열람이 차단됨을 안내받았습니다.

AI가 정리한 사건 요약과 문서 초안의 기초는 법률 자문이나 최종 문서가 아니며, 최종 법률 판단과 문서 사용 여부는 반드시 변호사 또는 적법한 전문가의 검토를 거쳐야 함을 확인합니다.
```

## 5. 변호사 열람 고지문

변호사가 사건 패키지를 열람할 때 아래 고지문을 기준으로 한다.

```text
본 사건 패키지는 의뢰인이 공유한 사건 정리 자료입니다. AI가 정리한 사건 요약, 인터뷰 요약, 문서 초안의 기초는 법률 자문이나 최종 문서가 아니며, 변호사의 독립적인 검토와 판단을 대체하지 않습니다. 본 자료에는 의뢰인이 허용한 범위의 정보만 포함되며, 공유 범위 밖의 첨부파일 원문, 내부 prompt, 모델 raw response, snapshotJson 전체는 제공되지 않습니다. 사건 패키지 열람 및 다운로드 기록은 보안과 분쟁 예방을 위해 로그로 저장될 수 있습니다.
```

## 7. 로그 저장 기준

사건 패키지 접근 로그는 아래 기준으로 저장한다.

### 로그 대상

- 사건 패키지 조회
- 사건 패키지 상세 열람
- 첨부파일 다운로드
- 사건 패키지 요약본 다운로드
- 접근 차단
- 만료 차단
- 취소 차단

### 로그 action

- VIEW
- DOWNLOAD
- DENIED
- EXPIRED
- REVOKED

### 로그 targetType

- SUMMARY
- INTERVIEW
- ATTACHMENT_LIST
- ATTACHMENT
- DOCUMENT
- PACKAGE

### 로그 보관 목적

- 의뢰인의 공유 이력 확인
- 변호사 열람 이력 확인
- 다운로드 이력 확인
- 분쟁 예방
- 보안 감사
- 접근 차단 사유 확인

### 로그 표시 기준

의뢰인 화면에서는 아래 항목만 표시한다.

- action
- targetType
- targetId
- resultMessage
- createdAt
- actor name/email 후보
- actorUserId 후보

아래 항목은 일반 UI에 직접 노출하지 않는다.

- IP 전체
- User-Agent 전체
- 내부 stack trace
- secret
- token/hash
- raw payload

## 8. 변호사법 오인 방지 문구

AI법친은 변호사 또는 법률전문가를 대체하지 않는다.

### 기본 문구

```text
AI법친은 변호사를 대체하지 않습니다. AI법친이 제공하는 사건 요약, 질문 정리, 문서 초안의 기초, 사건 패키지 요약본은 법률 자문이나 최종 법률문서가 아닙니다. 최종 법률 판단, 전략 수립, 문서 제출 여부, 수사·재판 대응은 반드시 변호사 또는 적법한 전문가의 검토를 거쳐야 합니다.
```

### 금지 표현

- "AI가 법률상담을 제공합니다."
- "AI가 고소장을 완성합니다."
- "변호사 없이 해결 가능합니다."
- "승소 가능성을 보장합니다."
- "법률 판단을 자동으로 확정합니다."
- "AI 변호사가 검토합니다."

### 허용 표현

- "사건을 정리합니다."
- "변호사 검토 전 자료를 준비합니다."
- "질문 흐름에 따라 사실관계를 구조화합니다."
- "문서 초안의 기초를 제공합니다."
- "최종 검토는 변호사가 수행합니다."
